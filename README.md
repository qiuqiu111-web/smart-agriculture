# 智慧农业系统

基于 STM32 + LubanCat 双板架构的智慧农业数据采集与控制系统。

## 硬件架构

```
STM32F4 (传感器节点)  ←——UART——→  LubanCat (边缘计算网关)
```

### STM32F4 — 传感器采集与设备控制

负责底层硬件交互：

- 多传感器数据采集（温湿度、土壤传感器等）
- 水泵远程控制
- FreeRTOS 多任务调度
- 通过 UART 协议与上位机通信

详见 [stm32f4/](stm32f4/)

### LubanCat — 边缘计算与数据处理

负责上层数据处理与业务逻辑：

- 接收 STM32 传感器数据（UART）
- 数据流水线处理与特征提取
- 控制策略执行
- CSV 数据落盘
- JSON 配置文件驱动

详见 [lubancat/](lubancat/)
